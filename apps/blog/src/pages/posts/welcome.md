# Welcome to Minions Content-research

We're excited to introduce **Minions Content-research** — Research sources, topic clusters, keyword sets, and content briefs.

## What's Included

- **TypeScript SDK**: `npm install @minions-content-research/sdk`
- **Python SDK**: `pip install minions-content-research`
- **CLI**: `npm install -g @minions-content-research/cli`

Check out the [documentation](https://content-research.minions.help) for a full guide.
