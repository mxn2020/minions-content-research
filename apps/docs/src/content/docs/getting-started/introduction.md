# Minions Content-research

Research sources, topic clusters, keyword sets, and content briefs

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-content-research/sdk`) — Core library for Node.js
- **Python SDK** (`minions-content-research`) — Python bindings
- **CLI Tool** (`@minions-content-research/cli`) — Command-line interface
- **Web App** — Interactive playground at [content-research.minions.wtf](https://content-research.minions.wtf)
